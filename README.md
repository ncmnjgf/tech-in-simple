## 🚀 About The Project

**Tech In Simple** is a lightning-fast, highly-opinionated web application that utilizes Google's Gemini 2.5 Flash AI to explain complex technical concepts across five distinct comprehension levels (Kid, Student, Interview, Analogy, One-Liner). 

Built entirely upon a stark, performance-driven **Brutalist UI pattern**, the application forces clarity not just in its explanations, but in its visual language. 

### Why This Exists?
Modern tech documentation is convoluted. This app aggressively cuts the noise and adapts to *your* preferred learning persona—whether you need a formal teacher, a witty comedian, or a casual friend.

## ✨ Core Features

* 🎭 **Persona Engine:** Dynamically injects stylistic personas (Teacher, Friend, Funny, Concise) directly into the AI generation flow.
* 🌐 **Global Translation Generator:** Generate fluent multi-level explanations in English, Spanish, Hindi, Japanese, French, and German.
* 🧠 **"Ask Why?" Deep Dive:** Bespoke elaboration logic that hits Gemini to crack open the hood on any explanation paragraph of your choice.
* 🪄 **Real-time UX Rewrites:** Effortlessly rewrite specific cards to "Make Simpler" directly within the responsive Brutalist grid.
* 🗄️ **Zero-Config Excel Database:** Features a fully native local `Excel` database architecture using `xlsx` to handle secured JWT Auth & Saved History flawlessly without external database dependencies.
* 📱 **Mobile-First Layout:** Fluid CSS grid layers styled uniquely for aggressive Brutalism architecture across all devices.

## 🛠⚙️ Quick Start Installation

Run this project seamlessly on your local machine.

### Prerequisites
* [Node.js](https://nodejs.org/en/) installed on your machine.
* A free API key from [Google AI Studio (Gemini)](https://aistudio.google.com/app/apikey).

### Setup

1. **Clone the repository**
   ```sh
   git clone https://github.com/ncmnjgf/tech-in-simple.git
   cd tech-in-simple
   ```

2. **Configure Environment Variables**
   Create a `.env` file inside the `/backend` folder:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   PORT=5000
   JWT_SECRET=super_secret_brutalism_key_123
   ```

3. **Install Dependencies & Run Backend**
   ```sh
   cd backend
   npm install
   npm run dev
   ```

4. **Install Dependencies & Run Frontend (Optional for React Devs)**
   *Note: The backend automatically unpacks and hosts the compiled frontend production build for extreme portability.*
   ```sh
   cd ../frontend
   npm install
   npm run dev
   ```

## 🖤 Design Philsophy (Brutalism)

The UI was meticulously architected by entirely bypassing typical modern SaaS aesthetics (rounded corners, soft shadows). Instead of fighting for attention, the UI utilizes stark `border: 4px solid #000`, absolute monochrome layers (`#000` / `#fff`), and sharp interactivity models `translate(-2px, -2px)` backed by heavy hard-edged shadows to yield extremely high perceptual interactivity.
